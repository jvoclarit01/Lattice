Lattice TUI: Core Features

Select all features you want included in the initial version of the TUI

1

DPEV Loop Interactive Wizard

Forms to capture Discuss phase decisions, run verification checklist items, and automatically write CONTEXT.md, PLAN.md, and VERIFICATION.md in the active phase folders.

2

Discipline Skill Browser & Search

Searchable quick-reference viewer for all 66+ Lattice skills (webdev, ML, thesis, shared). Read Iron Laws, decision tables, and checklists directly in the terminal.

3

Project Dashboard & Task Manager

Displays active mode, active phase, and tasks from .lattice-plan.md with interactive status toggles (Uncompleted, In Progress, Completed).

4

Graphify Dashboard & Query Tool

Run graph builds or incremental updates, view community stats, and query the codebase's knowledge graph (BFS/DFS traversal paths) directly inside the TUI.

5

Forge to Lattice Migration Tool

Graphical interface for the migrate-from-forge.ps1 script, letting you dry-run or migrate legacy projects visually.
